How it works

어떻게 작동하나요?

기기에 동기화된(또는 온라인) 날씨의 풍향·풍속과, GPS로 측정한 내 진행 방향을 매초 비교합니다.

🧭

진행 방향

GPS course(heading)를 실시간으로 읽어 화면 위쪽을 항상 "내가 가는 방향"으로 고정합니다.

🌬️

바람 + 기온

풍향(불어오는 방향)·풍속과 기온·체감온도를 가져와 라이더 기준으로 변환합니다.

➡️

충돌 시각화

상대 각도로 맞/뒷/옆바람을 분해. 화살표 방향·색·길이로 영향의 세기를 즉시 전달합니다.

↓ 빨강 맞바람(저항) ↑ 초록 뒷바람(가속) 대각 주황 옆+약한 맞바람 화살표 길이 = 바람 세기

How to use

사용법 (설치 → 라이딩)

처음 한 번만 설정하면 됩니다. 약 5분.

1. 설치

Connect IQ Store / Garmin Connect 앱에서 Wind Field 검색 후 설치. 설치 즉시 14일 트라이얼(전 기능)이 시작됩니다.

2. 데이터 화면에 추가

Edge에서 활동 프로필 → 데이터 화면 → 필드 추가 → Connect IQ → Wind Field. 화살표를 보려면 풀스크린 또는 큰 필드에 배치하세요.

3. 날씨 동기화

라이딩 전 휴대폰 Garmin Connect 앱과 동기화(또는 Wi-Fi). 그래야 풍향·풍속·기온이 채워집니다. 안 되면 화면에 NO WIND DATA 표시.

4. (선택) 온라인 날씨 켜기 — 프리미엄

더 신선한 데이터를 원하면 설정에서 Online weather를 켜고 OpenWeatherMap API Key를 입력. 위치 기반으로 15분마다 갱신·캐시합니다.

5. 라이선스 키 입력 (구매 후)

Garmin Connect 앱 → 기기 → Connect IQ → Wind Field → 설정 → License Key 에 키(예: WF-XXXXXX-YY) 입력 후 동기화 → 프리미엄 영구 적용.

6. 라이딩 시작

야외에서 GPS 고정 + 이동 시작 → heading이 잡히면 화살표가 살아납니다. 위쪽이 내 진행 방향, 화살표가 바람.

① 우선순위

온라인 캐시가 신선(<15분)하면 온라인을 쓰고, 없거나 오래됐으면 기기 내장 Weather로 폴백합니다. 화면 하단 online/cache/device 배지로 현재 소스를 표시.

② 풍향 규약

두 소스 모두 풍향은 "바람이 불어오는 방향"(기상학 표준, 0°=북풍). 앱이 진행 방향과 비교해 relAngle = 풍향 − heading로 맞/뒷/옆을 분해합니다.

③ 비용/키

온라인은 OpenWeatherMap 무료 키(분당 60회)면 충분. 캐시·throttle로 호출을 최소화합니다. 키 없이도 ① 기기 내장만으로 완전히 동작합니다.

자주 묻는 질문

바람·기온 데이터는 어디서 오나요?

기본은 Garmin 기기에 동기화된 Weather 정보, 프리미엄에서 OpenWeatherMap 온라인을 위치 기반으로 추가 사용합니다. 자세한 비교는 위 "데이터 출처" 표 참고.

온라인 날씨를 쓰면 데이터/배터리를 많이 먹나요?

아니요. 15분 캐시 + 3km 이동 시에만 갱신 + 재요청 최소 60초로 호출을 최소화합니다. JSON 응답도 작습니다.

어떤 기기에서 동작하나요?

현재 Edge 1030 Plus 대상입니다. Edge 1040·840 등도 손쉽게 추가 가능(Connect IQ 3.2.0+).

트라이얼이 끝나면?

설치 후 14일간 전 기능 개방. 만료되면 무료 기능(맞/뒷바람 화살표 + 수치)으로 돌아가고, 라이선스 키 입력 시 즉시 영구 복원됩니다.

구독인가요?

아니요. 한 번 결제하는 평생 라이선스입니다. 자동 결제가 없습니다.

라이선스 키는 어떻게 입력하나요?

Garmin Connect 앱 → 기기 → Connect IQ → Wind Field → 설정 → License Key 칸에 키 입력 후 동기화.
