자주 묻는 질문

궁금한 점

바람 데이터는 어디서 오나요?

기본은 기기에 동기화된 Garmin 예보 데이터(Weather API)입니다. 신호가 없어도 동작하지만 예보 기반이라 국지적 돌풍과는 차이가 있을 수 있습니다. 프리미엄에서는 본인 OpenWeatherMap API 키로 실시간 관측값을 더 자주 갱신할 수 있습니다.

어떤 기기에서 쓸 수 있나요?

현재 Garmin Edge 1030 Plus를 공식 지원합니다(282×470 해상도 최적화). 다른 Edge 모델은 manifest에 product id를 추가해 확장할 수 있으며, Weather API 특성상 Connect IQ 3.2.0 이상이 필요합니다.

정지하거나 느릴 때는요?

진행 방향(heading)은 GPS 코스 기반이라 정지·저속에서는 부정확할 수 있습니다. 이때는 마지막 유효 값을 유지하며, GPS가 잡히지 않으면 "NEED GPS / MOVE"를 표시합니다.

구매한 키를 다른 기기에서도 쓸 수 있나요?

라이선스 키는 오프라인 검증 방식이라 본인 소유 기기에 동일 키를 입력해 사용할 수 있습니다. 키는 Garmin Connect 앱 설정에 한 번만 입력하면 됩니다.

환불이 되나요?

14일 무료 트라이얼로 모든 프리미엄 기능을 먼저 체험할 수 있어, 구매 전 충분히 확인 가능합니다. 결제 후 환불은 사용한 외부 결제 플랫폼 정책을 따릅니다.
