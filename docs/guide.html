바람 데이터 필드 (Wind Field)

가민 사이클링 컴퓨터를 위한 실시간 풍향/풍속 분석 및 에어로 효과 측정 솔루션

버전 1.0.0
Edge 530 / 1030 Plus 지원
Connect IQ 3.2.0+

소개 (Introduction)

바람 데이터 필드 (Wind Field)는 가민 자전거 속도계의 공식 날씨 API와 GPS 센서를 연동하여 라이더가 주행하는 도중 실시간으로 어떤 방향에서 어떤 속도의 바람을 맞고 있는지를 직관적으로 보여주는 프리미엄 데이터 필드입니다.

단순히 풍향과 풍속을 숫자로 보여주는 것을 넘어, 자전거 주행 역학(Aero Drag dynamics)을 적용한 에어로 저항 변화 수치 및 순풍을 타고 나아가는 돛(Sailing) 효과 검출 기능을 제공하여 더 효율적인 페이싱과 기어비를 선택할 수 있게 돕습니다.

핵심 기능 (Key Features)

동적 360° 풍향계

나의 진행 방향을 기준으로 상대적인 바람이 불어 나가는 방향을 나침반 원반과 화살표로 실시간 표시합니다.

에어로 드래그 지수

상대적 기류 각도(Yaw) 및 풍속 데이터를 자전거 에어로 다이나믹 계수(CdA)에 대입해 맞바람 저항 증감율(%)을 실시간 계산합니다.

돛 효과 (SAIL) 검출

기류 각도가 자전거를 뒤나 옆에서 앞으로 밀어주는 각도(Yaw 45°~135°)에 있을 때 에어로 저항이 보너스로 작용하는 돛 효과 상태를 식별합니다.

자동 화면 레이아웃

자전거 컴퓨터 화면 분할 칸 크기(높이)에 따라 최적의 레이아웃(Full, Mid, Compact)으로 정보 밀도를 자동 튜닝합니다.

설치 및 셋업 가이드

1

Connect IQ 스토어 검색 및 다운로드

휴대폰의 Garmin Connect IQ Store 앱을 켜고 "Wind Field" 또는 "바람 데이터 필드"를 검색하여 다운로드 및 설치합니다.

2

데이터 화면 구성에 데이터 필드 추가

가민 본체에서 [설정] > [활동 프로필] > 프로필 선택 > [데이터 화면] > 화면 선택 > 데이터 필드 추가 > [Connect IQ] 카테고리에서 **Wind Field**를 선택하여 배치합니다.

3

날씨 데이터 동기화 확인

가민 컴퓨터는 자체 인터넷 센서가 없으므로 연동된 휴대폰의 Garmin Connect 앱을 통해 날씨 동기화가 필요합니다. 동기화가 실패할 시 화면에 `NO WIND DATA`가 노출됩니다.

4

GPS 확인 및 주행 시작

방향을 제대로 분석하려면 자전거가 움직여 GPS 진행 방향(Heading)이 제공되어야 합니다. 주행을 시작하면 자동으로 화살표가 동작합니다.

주의사항 (Warning)

바람의 절대 각도는 가민 연동 날씨 스테이션의 최근 정보를 기초로 합니다. 따라서 급격한 산악 지형이나 장애물이 많은 도심지에서는 실제 느끼는 바람과 예보상의 바람 방향 차이가 발생할 수 있습니다.

설정 및 라이선스 등록

라이선스 키를 등록하여 무료 체험판(Trial 14일) 제한을 풀고 프리미엄 버전을 영구히 활성화할 수 있습니다.

1. 휴대폰의 [Garmin Connect] 앱 실행 2. [장치] > [Connect IQ 앱] > [데이터 필드] > [Wind Field] 선택 3. [설정 (Settings)] 버튼 터치 4. 'License Key' 설정 항목에 구매한 키 입력 (형식: WF-XXXXXX-YY) 5. 가민 기기 동기화 진행

설정이 올바르게 완료되면 데이터 필드 우상단의 `TRIAL` 또는 `PRO?` 배지가 사라지고 전체 에어로 분석 기능이 평생 잠금 해제됩니다.
